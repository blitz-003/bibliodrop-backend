# BiblioDrop Backend 🚀

The backend service for **BiblioDrop**, providing RESTful APIs for authentication, book management, orders, payments, and user operations.

## 🌐 Live API

**API:** https://bibliodrop-backend.vercel.app

## 🔗 Repository

**GitHub:** https://github.com/blitz-003/bibliodrop-backend

## ✨ Features

- RESTful API
- JWT Authentication & Authorization
- User and Admin roles
- Book CRUD operations
- Book approval workflow
- Order and payment processing
- Image upload support
- MongoDB database integration

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Stripe
- Cloudinary

## 🚀 Getting Started

```bash
git clone https://github.com/blitz-003/bibliodrop-backend.git
cd bibliodrop-backend
npm install
npm start
```

The server will run on `http://localhost:5000` (or the port specified in your `.env` file).

## Precaution

First set up environment variables in your .env file
